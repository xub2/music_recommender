# 🤔개발 목적

**내가 좋아 하는 가수의 비슷한 노래를 추천 받아 볼 수 없을까? 라는 생각에서 프로젝트를 계획**

- 저번 학기 DB응용 수업에서 감정 분석을 통한 음악 추천에서 파생된 주제.
- 내가 좋아하는 가수를 입력하면 노래를 추천 받을 수 있는 어플을 개발 해보고 싶었음.
- 개인이 할 수 있는 난이도의 프론트 + 백엔드 작업을 직접 학습해 볼 수 있는 프로젝트라 생각되어 선택하게 되었음.
